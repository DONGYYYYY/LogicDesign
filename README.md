> 현대 디지탈 논리학의 역사적 배경과 기호논리학 및 부울대수학을 강의한다. 부울 대수학 이론을 바탕으로 한 회로 대수학 및 디지털 논리학을 강의하고 디지털 논리에 의한 디지털 회로의 구현에 대해서 세부적으로 살펴본다. ```"조합회로"```와 ```"순차회로"```의 구분에 대해서 살펴보고, 각 회로를 효과적으로 최적화 하여 설계하는 방법에 대해서 강의한다.
